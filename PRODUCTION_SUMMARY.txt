ENVIRONMENT VARIABLES NEEDED
================================================================================

For Local Development (.env.local):
  VITE_SUPABASE_URL=[your-supabase-url]
  VITE_SUPABASE_ANON_KEY=[your-anon-key]

For Vercel Production:
  Same variables via Vercel dashboard
  Settings → Environment Variables → Add VITE_SUPABASE_URL
  Settings → Environment Variables → Add VITE_SUPABASE_ANON_KEY

The Supabase credentials are currently embedded in src/lib/supabase.js as fallback
values. For production, move them to environment variables.
